# OneUp Admin

A branded WordPress admin experience for OneUp Motion.

## What it does

- Styles the WordPress admin to match the OneUp Motion visual direction.
- Adds a branded login page.
- Adds a custom dashboard with useful quick links.
- Cleans up the admin menu.
- Can hide cookie plugin menu items without disabling the cookie plugin.
- Adds a settings page under **OneUp Admin**.

## Installation

1. Upload the `oneup-admin` folder to `wp-content/plugins/`.
2. Go to **Plugins** in WordPress.
3. Activate **OneUp Admin**.
4. Go to **OneUp Admin** in the WordPress menu.

## Cookie plugin hiding

The plugin only hides matching menu items in the WordPress admin. It does not deactivate the cookie plugin and does not remove the cookie banner on the frontend.

Default keywords include:

- complianz
- cmplz
- cookie
- cookies
- cookie-notice
- cookie-law
- iubenda
- cookiebot
- gdpr
- borlabs

You can add extra keywords in **OneUp Admin > Menu**.

## Notes

This plugin does not edit WordPress core and does not replace the frontend theme.
